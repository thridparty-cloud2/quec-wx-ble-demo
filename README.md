# 微信小程序蓝牙配网Demo

本项目演示了基于 Quec 小程序插件的 BLE 一键配网与设备绑定流程，当前版本包含登录页和配网页两部分。

## 能力

- 账号登录（手机号/邮箱 + 密码）
- 获取当前手机 WiFi SSID
- 开启蓝牙与定位能力
- 扫描并选择 BLE 设备
- 一键配网（连接、开通知、发 WiFi、监听状态）
- 配网成功后自动轮询绑定设备
- 过程日志输出与一键重置配网

## 快速开始

1. 打开微信开发者工具。
2. 选择“导入项目”，项目目录指向本项目根目录（demo 文件夹）。
3. AppID 选择“测试号”或填写你自己的小程序 AppID。
4. 在微信公众平台给小程序添加 Quec 插件，并确认插件权限。
5. 在 `app.json` 中确认插件配置与版本号。

当前 `app.json` 中插件配置示例：

```json
"plugins": {
  "quecPlugin": {
    "version": "3.0.0",
    "provider": "wx8b0625890b89be58"
  }
}
```

## 使用流程

### 1. 登录

打开小程序后先进入登录页：

- 输入手机号或邮箱
- 输入密码（至少 6 位）
- 点击登录，成功后自动跳转到配网页

### 2. 配网三步

在配网页按以下顺序操作：

1. 填写 WiFi 信息
2. 开启蓝牙并开始扫描，选择目标设备
3. 点击“开始配网”执行一键流程

一键配网内部流程：

1. 连接设备（connectBLE）
2. 开启通知（openNotifyBLE）
3. 注册状态回调（onBLECharacteristicValueChangeV2）
4. 发送 WiFi 信息（writeBLECharacteristicValue）
5. 收到成功状态后轮询绑定（bindDeviceByWifi）

## 配网状态码

| 状态码 | 说明         |
| ------ | ------------ |
| 0x0010 | 路由器扫描中 |
| 0x0020 | 路由器连接中 |
| 0x0030 | 云平台认证中 |
| 0x0040 | 云平台登录中 |
| 0x0050 | 云平台鉴权中 |
| 0x0060 | 绑定中       |
| 0xfffe | 配网成功     |
| 0xffff | 配网失败     |

## 真机调试

蓝牙功能在模拟器中不可用，请使用真机调试：

1. 点击微信开发者工具右上角“真机调试”。
2. 使用手机微信扫码运行预览。
3. 确保手机蓝牙和定位权限已开启。

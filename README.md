# 蓝牙配网流程 Demo 小程序

这是一个独立的微信小程序 Demo，演示了通过蓝牙为 IoT 设备配网的完整流程。

## 快速开始

1. 打开 **微信开发者工具**
2. 选择「导入项目」，项目目录指向本 `demo/` 文件夹
3. AppID 选择「测试号」或填入你自己的小程序 AppID

## 配置插件

在 `app.json` 中找到 `plugins.quecPlugin.provider`，填入你的插件 AppID：

```json
"plugins": {
  "quecPlugin": {
    "version": "3.0.0",
    "provider": "wx8b0625890b89be58"
  }
}
```

> 如果尚未申请插件，请先在微信公众平台的小程序后台搜索并添加 Quec 插件。

## 真机调试

蓝牙功能在模拟器中不可用，请使用 **真机调试**：

1. 点击开发者工具右上角「真机调试」
2. 用手机微信扫码预览
3. 确保手机蓝牙和定位已开启

## 配网流程概述

按页面上的编号顺序操作：

| 步骤 | 操作 | 说明 |
|------|------|------|
| ① | 获取 WiFi | 获取当前手机连接的 WiFi 名称 |
| ② | 开启蓝牙 | 打开蓝牙和定位权限 |
| ③ | 扫描设备 | 搜索附近的蓝牙配网设备 |
| ④ | 停止扫描 | 停止设备搜索 |
| ⑤ | 连接设备 | 连接选中的蓝牙设备 |
| ⑥ | 开启通知 | 开启 BLE Notify 特征值通知 |
| ⑦ | 监听结果 | 开始监听配网状态回调 |
| ⑧ | 发送 WiFi | 将 WiFi 名称和密码发送给设备 |
| ⑨ | 绑定设备 | 配网成功后绑定设备到账号 |
| ⑩ | 断开 / 关闭 | 断开蓝牙连接并关闭蓝牙模块 |

### 状态码对照

| 状态码 | 含义 |
|--------|------|
| 0x0010 | 路由器扫描中 |
| 0x0020 | 路由器连接中 |
| 0x0030 | 云平台认证中 |
| 0x0040 | 云平台登录中 |
| 0x0050 | 云平台鉴权中 |
| 0x0060 | 绑定中 |
| 0xfffe | 配网成功 |
| 0xffff | 配网失败 |
